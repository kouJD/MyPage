angular 过滤器

过滤器可以使用一个管道字符（|）添加到表达式和指令中

angular过滤器可以用来转换数据，方法有：

过滤器 描述
currency 格式化数字为货币格式
filter 从数组项中选择一个子集(可用在筛选、查找)
lowercase 格式化字符串为小写
orderBy 根据某个表达式排列数组
uppercase 格式化字符串为大写

表达式中添加过滤器

过滤器可以通过一个管道字符（|）和一个过滤器添加到表达式中

uppercase过滤器将字符串格式化为大写：


	
{{ lastName | uppercase }}
输出为：

{{ lastName | uppercase }}

currency格式化数字为货币格式（自动在前面添加$符，并且自动添加小数点什么的）


	单价：				
	总价：
	
需付：{{ (quantity*price) | currency }}
输出为：
单价： 总价：

需付：{{ (quantity*price) | currency }}
